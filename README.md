# PROJECT-PENGENALAN-PEMROGRAMAN
program absen sederhana yang akan memiliki beberapa fitur utama sebagai solusi dari permasalahan di atas:

1. Input Data Absensi – pengguna dapat memasukkan nama dan status kehadiran (H/T). Data otomatis disimpan ke file absensi.txt.

2. Tampilkan Daftar Absen – data dibaca dari file dan ditampilkan dalam urutan abjad.

3. Cari Nama – pengguna dapat mencari nama tertentu untuk melihat statusnya.

4. Rekap Kehadiran – menampilkan jumlah siswa yang hadir dan tidak hadir.

5. Ubah Status – memperbarui data kehadiran siswa jika terjadi kesalahan.
